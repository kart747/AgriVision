1. GEO-AWARE LLM
   Same disease → different recommendations based on farmer's location
   Implemented: LLM_PROMPT_DESIGN.md (Architecture section)
   
2. EXPLAINABLE AI (Grad-CAM)
   Shows WHERE model looked to make decision
   Builds farmer trust vs. black box
   Implemented: backend/model/gradcam.py
   
3. DUAL-LAYER SAFETY
   Confidence gate (60%) filters uncertain predictions
   Knowledge base fallback if LLM unavailable
   Implemented: backend/main.py + llm_validation/advisor.py
   
4. GPS EXIF PARSING
   Auto-fills location from photo metadata
   Enables location-aware recommendations without extra user input
   Implemented: frontend/detect.html (piexifjs library)
   
5. FARMER-FIRST UX
   No ML jargon. Just upload → see recommendation.
   Designed for field use, not research lab
   Implemented: frontend/detect.html
